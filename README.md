# Android Horizon — website

Source for the Android Horizon project website — a static site, no build step. Plain HTML/CSS in `index.html`, `compatibility.html`, `docs/index.html`, and `assets/`.

To preview locally, just open `index.html` in a browser, or serve the folder with any static file server (e.g. `python3 -m http.server`).

Deployed via GitHub Pages from this repo's `main` branch.
